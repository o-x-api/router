# Security Policy

We take security seriously. If you discover a vulnerability or have security concerns, please report them responsibly.

---

## 🛡️ Supported Versions

We actively support and apply security patches to the following versions of **GH_OmniRout**:

| Version | Supported |
| :--- | :---: |
| Active `main` branch | ✅ Yes |
| All prior versions | ❌ No |

---

## 🔒 Reporting a Vulnerability

**Please do not report security vulnerabilities via public GitHub issues.**

Instead, please report security vulnerabilities directly to the project maintainers. 
* To report a vulnerability, open a private vulnerability report via the **Security** tab of this repository on GitHub.
* We will investigate and respond to all reports within 48 hours.
* We will coordinate public disclosures with you once the vulnerability is successfully mitigated.
